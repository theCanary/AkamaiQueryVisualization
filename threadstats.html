Thread Statistics over Time

This data is pulled from the TST pipeline, which logs thread usage statistics for each thread. A note about the table structure: the column with the CPU time is considered the sum of the UserTime and SystemTime columns, so the total time is the same as the CPU time.

Query Settings

Type in the Network and Domain boxes to narrow down the list of possible aggsets, and select the field you want to limit your query by. Note on the threads list: Certain threads such as 'decoder' are multithreaded and appear in the logs as 'decoder01', 'decoder02', etc. For our purposes all of these have been merged together and their values will be the average of each component value.

Queries may take over 30 seconds to render, so please be patient!
